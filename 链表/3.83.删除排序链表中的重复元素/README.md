# 3(83) 删除排序链表中的重复元素
## 描述
给定一个排序链表，删除所有重复的元素，使得每个元素只出现一次。
## 示例
```bash
输入: 1->1->2
输出: 1->2

输入: 1->1->2->3->3
输出: 1->2->3
``` 

## Description
