# 9(145) 二叉树的后序遍历(Hard)
## 描述

给定一个二叉树，返回它的 后序 遍历

## 示例
```bash
输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [3,2,1]
```
